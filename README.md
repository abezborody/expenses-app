# expenses-app

[Web demo](https://expenses-app-demo.fly.dev/)

### Stack:

- TypeScript
- Bun
- Hono
- Vite
- React
- Tanstack
  - Router
  - Query
- Shadcn
  - Tailwind
- Zod

To install dependencies:

```bash
bun install
```

### To run for dev:

```bash
bun dev
```

cd to frontend folder and run

```bash
bun run dev
```

### To build run in frontend folder:

```bash
bun run build
```
